# 使い方
1. ファイルをダウンロードする
```
git clone https://github.com/mami-inuzuka/vue_note_spa.git
```

2. プロジェクトファイルに移動
```
cd vue_note_spa
```

3. サーバーを起動
```
npm run serve
```

4. 下記URLにアクセス

http://localhost:8080/
